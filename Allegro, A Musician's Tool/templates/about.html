Why Allegro?

This wesbite was created specifically for musicians. Finding sheet music can be a pain sometimes, and we understand that struggle. So, for our Multimedia Design and Programming class, we wanted to implement such an idea to help musicians. The website includes features to convert live audio, or a given file. Our program will then analyze the audio and return a text file with the appropriate notes and octaves. The user will be able to download this new file or email it to themselves. The website also includes a tuning feature, where given live audio, will display the musical note that is being received. Moreover, the website includes features to give user feedback, and display user feedback, while also being able to remove the feedback if desired. Finally, the website includes a suggestions section where users can send us emails based on comments or suggestions they have for our work.

About Us
